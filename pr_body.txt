This PR integrates web research capabilities into z-code:

- **web_search**: Implements DuckDuckGo search to allow the agent to find latest API documentation and troubleshooting guides.
- **web_fetch**: Implements webpage content extraction with noise removal (stripping scripts, nav, footer, etc.) to provide clean text for the LLM.
- **Infrastructure**:
  - Added `node-fetch` and `cheerio` dependencies.
  - Integrated tools into `src/tools/index.ts` and the tool registry.
  - Followed the prompt-separation architecture by adding corresponding `.txt` description files.

Verified with `npm run build`.

Exact string replacement in files.

Usage:
- MUST 'read' the file before editing.
- `oldString` must match exactly (including indentation).
- `oldString` must be unique in the file unless `replaceAll` is true.
- To ensure uniqueness, include more surrounding context lines in `oldString`.
- Use `replaceAll: true` for renames or file-wide changes.
- DO NOT add comments/emojis unless requested.

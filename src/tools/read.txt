Read text files or directories.

Usage:
- Use absolute paths.
- Returns up to 2000 lines. Use `offset` (1-indexed) to read further.
- Lines are prefixed with `<number>: `.
- Directory contents show entries one per line; subdirs end in `/`.
- Truncates lines >2000 chars.
- **Text only:** Errors on binary files.
- Call in parallel for multiple files.
